To use this package,

1. Edit line 2 of Install.sh to be the path to your LAMMPS installation
2. Run Install.sh.
3. Enable the package ( -D PKG_UQ-ROM=on )
4. Enable LAMMPS to download Eigen ( -D DOWNLOAD_EIGEN3=on )
5. Build LAMMPS as normal

See the README doc in the parent folder for more information about the different commands.
